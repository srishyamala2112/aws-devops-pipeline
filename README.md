# AWS Cloud-Driven DevOps Pipeline 🚀

## 📌 Overview
This project demonstrates a basic CI/CD pipeline built using GitHub Actions, Docker, and Terraform. The pipeline automates the process of building and testing a containerized Node.js application.

---

## ⚙️ Features
- Automated CI/CD pipeline using GitHub Actions
- Docker-based containerization of application
- Infrastructure provisioning using Terraform (AWS EC2)
- Continuous integration triggered on every code push

---

## 🛠️ Tech Stack
- AWS (EC2)
- GitHub Actions
- Docker
- Terraform
- Node.js

---

## 🔄 Workflow
1. Code is pushed to GitHub repository
2. GitHub Actions workflow is triggered
3. Docker image is built automatically
4. Container is executed to validate the application
