# AWS Cloud-Driven DevOps Pipeline

## Overview
Built a CI/CD pipeline using GitHub Actions, Docker, and Terraform.

## Features
- Automated build pipeline
- Docker containerization
- AWS EC2 provisioning using Terraform

## Tech Stack
- AWS
- GitHub Actions
- Docker
- Terraform
- Node.js

## Workflow
1. Code pushed to GitHub
2. GitHub Actions builds Docker image
3. Application runs inside container
